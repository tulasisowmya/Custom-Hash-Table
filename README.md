# 🔐 Custom Hash Table

This project demonstrates a basic custom hash table implementation in Python. It uses a simple character-based hash function and stores key-value pairs using dictionary-based buckets.

## 📌 Objectives

- Create a custom hash table structure.
- Generate hash values using character ASCII values.
- Store key-value pairs using hashed keys.
- Handle collisions using nested dictionary buckets.
- Implement add, lookup, and remove operations.

## 🛠️ Tools & Concepts Used

- Python
- Hashing
- Dictionaries
- Key-Value Storage
- Collision Handling
- Functions
- Object-Oriented Programming Basics
